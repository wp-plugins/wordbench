WordBench provides admin user interfaces for advanced WordPress features

== Description ==

WordBench provides admin user interfaces for advanced WordPress features:

*	Custom post types
*	Custom post formats
*	Custom taxonomies
* 	Custom roles & capabilities
	*	User-defined roles
	*	User-selected capabilities
	*	Multiple roles per user

WordBench is useful if you need to:

*	preserve custom post types and taxonomies when switching themes
*	add extra post formats to the fixed list provied by WordPress
*	assign users to multiple working groups for different data types

== Installation ==

1.	Upload the .zip file to your WordPress admin panel
1.	*OR* Unzip the .zip file and upload the `wordbench/` directory to the `/wp-content/plugins/` directory
1.	Activate the plugin through the 'Plugins' menu in WordPress
